Weather Forcasting Web-Application

This web-based application is a weather forecasting tool that allows users to retrieve real-time weather information for any city by entering the city's name in the search bar. The application communicates with the OpenWeatherMap API to fetch and display current weather data, including temperature, weather conditions, cloud cover, humidity, and atmospheric pressure. The interface is designed to be user-friendly, displaying weather data in a clear and visually appealing manner.

The application also includes error handling to manage invalid city names, providing feedback to the user when a city cannot be found.

Calculator

This web-based application is a simple yet functional calculator designed to perform basic arithmetic operations such as addition, subtraction, multiplication, and division. The calculator features a user-friendly interface that allows users to input numbers and operations, view results, and handle simple calculations with ease. It also includes additional functionalities like percentage calculation and a dark mode toggle.

Additionally, it supports keyboard inputs, allowing users to operate the calculator using either mouse clicks or keyboard shortcuts, which enhances the overall user experience.

Cricket Playground

A web-based application that brings the excitement of cricket to your fingertips. You can choose to play as the batsman, and compete against the computer's randomly generated choices. The computer's choices are generated using a random algorithm, ensuring a unique and challenging experience each time you play. With three options to choose from - Bat, Ball, and Stump.

Our application also keeps track of your score, with wins, losses, and ties all recorded and displayed on the screen. You can also reset your score at any time, and even clear your local storage to start fresh.
